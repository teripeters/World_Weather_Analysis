# World_Weather_Analysis

## Description
The purpose of this project is to create an app where users can create a vacation itinerary based on the user's preferred maximum and minimum temperatures. I began by creating a random list of cities by creating latitude and longitude pairs and then retreiving the nearest cities to the pairs. I then performed an API call with OpenWeatherMap to gather weather data for each city and create a dataframe. The dataframe was then exported to a csv file. In the next phase, we used the weather csv to create a dataframe and write two input statements to get the user's preferred minimum and maximum temperatures to filter their vacation options. Potential vacation locales were further filtered by those that have a hotel within 5000 meters of the city. Rows containing empty cells were removed and a new hotel dataframe and csv were created. I used gmaps to create a map with a marker layer and an info box for each location. In the third stage, I used gmaps to create a travel map beginning and ended in the same city and passing through three cities. The map also incluces info boxes.
